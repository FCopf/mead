# Modelagem Estatística e Análise de Dados

Material didático sobre Modelagem Estatística e Análise de Dados, com exemplos práticos em R e Python. Cobre desde conceitos fundamentais e análise exploratória até métodos avançados, incluindo inferência bayesiana, modelos lineares generalizados e estatística multivariada.

## Acesso ao material

Site do projeto:  
https://fcopf.github.io/MEAD/

Repositório do código fonte:  
https://github.com/FCopf/MEAD/

---

## Objetivo

Disponibilizar material didático com exemplos práticos em R e Python, abrangendo desde os fundamentos até métodos avançados de modelagem estatística e análise de dados.

---

## Estrutura do Projeto

- `content/` → Materiais didáticos escritos em Quarto (`.qmd`)
- `topics/` → Páginas de organização e navegação dos temas
- `images/` → Imagens e figuras utilizadas nos materiais
- `_site/` → Site gerado automaticamente pelo Quarto

---

## Tecnologias Utilizadas

- Quarto
- R
- Python
- Git e GitHub

---

## Execução local
